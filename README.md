# APIAutomation
 The same base framework is based on Rest Assured API .
 
 # Components of this framework
 1.Rest Assured JAR -  For initialting API calls ( Upload )
 2.Chrome driver & Gecko driver - For UI actions  ( Creating a token)
 3.Extent Report JAR - To generate Extent report for each test.
 4. Maven - To eliminate the build 
 # Hierarchy
  src/test/java/
 -> API.test  - Has the vairous test methods.
 -> genericLibrary - Classes with reusable methods for API  and UI.

  # Drivers
  Drivers required for UI login are placed under Common/Drivers.

  # Running the test
  Select the TestNg XML and run it.
